SPA(single page application) - приложение которое приходит один раз и может рендерит отдельные части страницы
SPA c помощью AJAX запросов может контролировать страницу.
Как происходит загрузка сайта:
1. Сначала клиент получает файл html(по сути полупустой) от сервера
2. Клиент делает ajax запрос и получает в ответ json
3. С помощью полученного json клиент заполняет страницу
4. Периодически клиент посылает запросы ajax на сервер для отрисовки изменений
